== README

* Collaborated in a team of 3 developers to create a DevOps web application to generate Bash scripts for deploying websites onto AWS EC2 instances.
* Stacks supported for deployment include MEAN, Python Django, Ruby on Rails.
* Features include full automation deployment for all supported stacks, user registration of students and instructors using an admin secret key, instructor/admin dashboard granting and removing access of student accounts, and auto-deletion of generated scripts upon user logout.
